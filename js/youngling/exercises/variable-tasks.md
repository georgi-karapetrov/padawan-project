# Variable exercises

### Old granny
Create a variable with name `granny` assign to the variable some text. Print the value in the console.
```js
const granny = "I'm not so old, m*fckaa"
console.log(granny)
```

### Sums and counts
Write the following in the console. What is the output?
```js
let count = 1
let sum = 3
sum = count + sum
count = sum + count
```
